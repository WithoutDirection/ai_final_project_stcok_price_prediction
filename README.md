# stock_price_prediction
* 訓練ai模型推估股票走勢

## 功能
* 從網路上爬取股票相關資訊
* 繪製股票走勢
* 利用該股票的歷史資訊訓練AI模型
* 預測未來股票走勢並決定是否買入、賣出、持有

## AI model
* random forest tree
* k-means
* LSTM
